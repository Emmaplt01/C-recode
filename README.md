# C-recode
Série d’exercices en langage C visant à re‑coder plusieurs fonctions de la bibliothèque standard, comme strlen, puts, etc... 

Ce répertoire est composé de plusieurs mini-fonctions, il faut les exécuter séparément.

Pour exécuter une mini-fonction (remplacer "nom" par la fonction que vous souhaitez):
gcc -Wall -Wextra -Werror -Iinclude nom.c nom.main.c
./a.out
